# Empowering everyone to build an endurance lifestyle.

Endurant is a platform for everyone in endurance sports, whether you are an elite, amateur, or lifestyle athlete.

At Endurant, we believe that becoming strong and healthy is a gateway to a better life, and endurance sports are the highest form of this pursuit. We exist to empower every athlete of all levels to be endurant through a lifestyle of progress, belonging, and joy. We do so through an AI-powered mobile application, also available via the web and smart watch.
